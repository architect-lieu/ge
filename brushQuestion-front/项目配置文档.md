# 基本配置

运行环境 node 18.21.0 +  npm 6.14.17

进入项目先安装依赖

```
npm install
yarn install
pnpm install
```

```
utils/config.js 中配置项目api

baseUrl：开发环境地址

proUrl：生产环境地址

appId: 你自己的小程序配置中的appId
```

## appid的获取
